## 常用的工具库

- 本地存储：storage cookie
- 日期：date2json dateFormat
- 网络请求：jsonp
- 单位换算：px2rem
- 碰撞检测：hitTestCircle hitTestContain hitTestRectangle
- 设备相关：getDeviceType getBroEnv getMobileType getNetwork

- 获取url参数：getUrlParameter
- 二叉树：BinaryTree
- 事件监听：EventEmiter
